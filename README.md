## Visualisation of the two-mode quantum state with two-mode Wigner function.

For two-mode state the Wigner function is defined as:

```math
\Gamma(n) = (n-1)!\quad\forall n\in\mathbb N
```

$$\sum_{i=0}^n i^2 = \frac{(n^2+n)(2n+1)}{6}$$


\\( \Gamma(n) = (n-1)!\quad\forall n\in\mathbb N \\)

$$\Gamma(n) = (n-1)!\quad\forall n\in\mathbb N $$
